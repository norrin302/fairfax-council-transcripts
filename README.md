# Fairfax City Council Transcripts

AI-powered transcription of Fairfax, VA City Council meetings.

## About

This project automatically transcribes public meeting videos from the City of Fairfax's Granicus archive, creating searchable, timestamped transcripts with speaker identification.

Created for **Councilwoman Stacy Hall** to support transparency in local government.

## Features

- 📹 Automatic video download from Granicus
- 🎤 AI transcription with OpenAI Whisper
- 👤 Speaker identification (Council members, City officials)
- ⏱️ Timestamped segments linked to video
- 🔍 Full-text search across all transcripts
- 📄 Multiple output formats (HTML, Markdown, WebVTT)

## Quick Start

```bash
# Install dependencies
pip install yt-dlp openai-whisper

# Transcribe a meeting
python scripts/transcribe.py "https://fairfax.granicus.com/player/clip/4421?meta_id=128481" --output .

# For API transcription (requires OPENAI_API_KEY)
python scripts/transcribe.py "<video_url>" --output . --api
```
